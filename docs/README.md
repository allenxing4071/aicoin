### 文档编号规则

```
格式: AICOIN-[类型]-[序号]

类型代码:
├─ PLAN     : 规划文档 (01-规划文档)
├─ ARCH     : 架构设计 (02-架构设计)
├─ DEV      : 开发指南 (03-开发指南)
├─ API      : API文档 (04-API文档)
├─ OPS      : 部署运维 (05-部署运维)
└─ RESEARCH : 研究报告 (06-研究报告)

示例:
- AICOIN-PLAN-001    : 01-规划文档/01-项目概述.md

```
版本号格式: vX.Y.Z

X (主版本号): 重大架构变更、文档重构

示例:
v1.0.0 → v1.1.0 (新增章节)
v1.1.0 → v1.1.1 (修正错误)

### AI交易金字塔

```
       🛡️  保护本金
            ↓
       📊  稳定收益
            ↓
      🎯  风险收益平衡
            ↓
     🤖  AI自主决策
```

### 五大原则

1. 🛡️ **资本保护优先** - 保护本金是第一要务
2. 📈 **长期主义** - 追求稳定收益，不追求暴利

4. 🧠 **持续学习** - AI从每次决策中学习

## 📊 技术栈

### 后端
- **框架**: Python 3.11 + FastAPI
- **数据库**: PostgreSQL 15 + Redis 7 + Qdrant
- **交易**: Hyperliquid SDK
- **AI**: OpenAI SDK + Transformers + PyTorch

### 前端
- **框架**: Next.js 14 + React 18 + TypeScript
- **样式**: TailwindCSS 3
- **图表**: Recharts + Lightweight Charts
- **通信**: Axios + Socket.IO

### 基础设施
- **容器**: Docker + Docker Compose
- **反向代理**: Nginx
- **云服务**: 阿里云 PAI-EAS（计划中）

## 🚀 快速部署

### Docker一键部署

```bash
# 1. 克隆仓库
git clone https://github.com/allenxing4071/aicoin.git
cd aicoin

# 2. 配置环境变量
cp .env.example .env
# 编辑.env文件，填入API密钥

# 3. 启动服务
docker-compose up -d

# 4. 访问前端
open http://localhost:3000
```

详见：[07-部署运维/部署指南.md](./07-部署运维/部署指南.md)

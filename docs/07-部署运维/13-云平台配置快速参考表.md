## ⚙️ 系统配置参数

### 环境变量配置

```bash
# Qwen (阿里云)
QWEN_API_KEY=sk-your-qwen-api-key
QWEN_BASE_URL=https://dashscope.aliyuncs.com/compatible-mode/v1
ENABLE_QWEN=true

# 百度千帆
BAIDU_QWEN_API_KEY=your_api_key:your_secret_key
BAIDU_QWEN_BASE_URL=https://aip.baidubce.com/rpc/2.0/ai_custom/v1/wenxinworkshop
ENABLE_BAIDU_QWEN=true

# 腾讯混元
TENCENT_QWEN_API_KEY=your_secret_id:your_secret_key
TENCENT_QWEN_BASE_URL=https://hunyuan.tencentcloudapi.com
ENABLE_TENCENT_QWEN=true

# 火山引擎
VOLCANO_QWEN_API_KEY=your_volcano_api_key
VOLCANO_QWEN_BASE_URL=https://ark.cn-beijing.volces.com/api/v3
ENABLE_VOLCANO_QWEN=true

# AWS Bedrock
AWS_QWEN_API_KEY=your_access_key:your_secret_key
AWS_QWEN_BASE_URL=https://bedrock-runtime.us-east-1.amazonaws.com
ENABLE_AWS_QWEN=true
```

## 🚨 常见错误代码

| 错误代码 | 含义 | 解决方法 |
|----------|------|----------|
| `401 Unauthorized` | API密钥无效 | 检查密钥是否正确、是否过期 |
| `403 Forbidden` | 权限不足 | 检查密钥权限、是否需要白名单 |
| `429 Too Many Requests` | 调用频率超限 | 降低调用频率或升级套餐 |
| `500 Internal Server Error` | 平台服务异常 | 等待平台恢复或切换其他平台 |
| `503 Service Unavailable` | 服务暂时不可用 | 稍后重试或切换其他平台 |
| `Connection Timeout` | 网络连接超时 | 检查网络、防火墙设置 |

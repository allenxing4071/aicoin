2. **L3卡片应该显示**:

## ❌ 如果有问题

### 问题1: Docker未运行
**现象**: 运行 `./start_all.sh` 提示 "Docker 未运行"  
**解决**: 打开Docker Desktop，等待它完全启动后再运行脚本

### 问题2: 端口已被占用
**现象**: 提示8000或3000端口已被占用  
**解决**:
```bash
# 停止现有进程
./stop_all.sh

# 重新启动
./start_all.sh
```

### 问题3: 前端显示旧数据
**现象**: 修改权限等级后，页面显示的还是旧的等级  
**解决**: 按 **Cmd+Shift+R** (Mac) 或 **Ctrl+Shift+R** (Windows/Linux) 硬刷新浏览器

### 问题4: 权限管理页面显示L1，但应该是L3
**现象**: 顶部显示"L1 新手级"  
**解决**: 等待10秒让页面自动刷新，或者按F5刷新页面

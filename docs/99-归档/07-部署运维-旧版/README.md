## 🔍 常见问题速查

### Q1: 我应该配置几个云平台？

**答案**：推荐至少配置3个平台（标准配置）

**原因**：
- 信息交叉验证，提升准确率
- 互为备份，提高可用性
- 成本可控，性价比最优

**参考**：[云平台配置快速参考表 - 推荐配置方案](./09-云平台配置快速参考表.md#推荐配置方案)

---

### Q2: 哪个平台最容易配置？

**答案**：Qwen (阿里云)

**原因**：
- 注册简单（支付宝登录）
- 免费额度充足
- 无需审核，即时开通
- 支持联网搜索

**参考**：[情报平台详细配置指南 - Qwen配置](./08-情报平台详细配置指南.md#1-qwen-阿里云---推荐首选)

---

### Q3: API密钥配置错误怎么办？

**答案**：查看图解说明，对照检查

**常见原因**：
- 密钥复制时包含空格
- 密钥格式不正确
- 密钥已过期

**参考**：[配置步骤图解说明 - 常见问题](./10-配置步骤图解说明.md#常见问题图解)

---

### Q4: 如何验证配置是否成功？

**答案**：查看平台状态和调用统计

**验证步骤**：
1. 平台状态显示"已启用"
2. 测试连接成功
3. 调用统计数据正常
4. 能看到新的情报报告

**参考**：[情报平台详细配置指南 - 配置验证](./08-情报平台详细配置指南.md#6-配置验证)

---

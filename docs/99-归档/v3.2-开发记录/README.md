## 🚀 部署信息

### 环境要求
- Docker: 20.10+
- Docker Compose: 2.0+
- PostgreSQL: 14+
- Node.js: 18+
- Python: 3.11+

### 部署步骤
```bash
# 1. 拉取最新代码
git pull origin main

# 2. 构建镜像
docker compose build

# 3. 启动服务
docker compose up -d

# 4. 检查服务状态
docker compose ps
```

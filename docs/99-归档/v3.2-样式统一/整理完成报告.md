## 🎨 最终样式规范

### 统计卡片样式
```css
bg-gradient-to-br from-{color}-50 to-{color}-100 
rounded-xl p-6 shadow-lg border-2 border-{color}-300
```

**特点**:
- 渐变彩色背景
- 阴影效果
- 更大圆角和内边距
- 更粗边框

### 列表卡片样式
```css
bg-white rounded-xl shadow-sm p-6 border-l-4 border-{color}-500
```

**特点**:
- 白色背景
- 左侧彩色边框
- 轻微阴影

### 按钮样式
```css
bg-gradient-to-r from-{color}-500 to-{color}-600
hover:from-{color}-600 hover:to-{color}-700
```

**特点**:
- 渐变背景
- 悬停效果
- 阴影增强

## ✨ 最佳实践总结

### 1. 组件化开发
```tsx
// ✅ 好的做法
import { StatCard } from '@/app/components/common/Cards';
<StatCard label="用户数" value={100} color="blue" />

// ❌ 避免的做法
<div className="bg-blue-50 border-blue-200 rounded-lg p-4">
  <div className="text-sm text-gray-600">用户数</div>
  <div className="text-2xl font-bold text-blue-600">100</div>
</div>
```

### 2. 颜色主题一致性
- 同一页面使用相关颜色
- 避免颜色混乱
- 遵循设计系统

### 3. 代码复用
- 优先使用组件库
- 避免重复代码
- 保持 DRY 原则
